# reading me
[마지막 페이지까지 정성을 다해 읽는 마음으로 일합니다]
저는 평소 책 읽는 시간을 소중히 여깁니다. 긴 호흡의 책 한 권을 끝까지 읽어내는 인내심은 제 생활 방식에도 고스란히 녹아있습니다. 한 곳에서 1년 넘게 꾸준히 아르바이트를 하며 제 자리를 지킬 수 있었던 것도, 맡은 일을 하나의 이야기처럼 소중히 여기며 마침표를 찍으려 노력했기 때문입니다. 일을 하다 보면 때로 예상치 못한 어려운 상황을 마주하기도 하지만, 그때마다 감정에 휘둘리기보다는 차분하게 상황을 살피고 합리적으로 해결하는 법을 익혔습니다. 화려하진 않아도 묵묵히 제 몫을 해내는 사람, 곁에 있으면 마음이 놓이는 든든한 동료가 되고 싶습니다.
