# reading me

저는 평소 책 읽는 시간을 소중히 여깁니다. 
긴 호흡의 책 한 권을 끝까지 읽어내는 인내심은 제 생활 방식에도 고스란히 녹아있습니다. 
화려하진 않아도 묵묵히 제 몫을 해내는 사람, 곁에 있으면 마음이 놓이는 든든한 동료가 되고 싶습니다.

평소 책을 가까이하며 사안을 객관적으로 살피는 습관을 길러왔습니다. 덕분에 긴장되는 상황에서도 중심을 잃지 않고 이성적으로 판단하려 노력하는 편입니다.
최근에는 많은 사람 앞에서 제 의견을 정리해 전달하는 발표를 무사히 마쳤습니다. 이를 통해 어떤 환경에서도 평정심을 유지하며 명확하게 소통하는 법을 익혔습니다. 누구에게나 예의를 갖추되, 부당한 상황에서는 논리와 근거를 바탕으로 제 목소리를 낼 줄 아는 단단함을 지향합니다.
유연하면서도 책임감 있는 태도로 제 역할을 다하겠습니다.
